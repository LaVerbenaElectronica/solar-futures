This v.1 of our website is not yet completely solar powered. But its design and hardware setup goes towards that goal. According to prior references of similar projects, the website should be hosted on a low powered device. In our case that device is an old Raspberry Pi B+.

The operating system that runs on this device is a Raspberry Pi OS (Legacy 32-bit) and inside there is a web server software lighttpd that serves pure static website.

Rapsberry Pi hardware runs on 5V 220mA, or about 0.9W. It is connected to a temporary solar power bank that needs to be charged during at nights. A more definitive version of the solar website, the v.2 is on its way.
